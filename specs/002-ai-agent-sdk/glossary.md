# Glossary

- AgentInstance: A single AI agent with model configuration, state, and execution context.
- WorkflowDefinition: A defined sequence of steps that an agent can execute autonomously.
- ModelConfiguration: Settings for AI models including API keys and model selection.
- ExecutionContext: Current lived context of an agent's activity.
- Real-time Protocol: WebSocket/SSE-based communication patterns.
- Latency: Time between sending a request and receiving a response. End-to-end latency includes network overhead.
- Acceptance Criteria: Concrete measurable conditions that must be satisfied for a user story or FR.
- Gate: A governance/approval checkpoint that must be satisfied before proceeding.
